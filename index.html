Graph API with clients
December 2013
written in Java. Wrote a library package to provide facilities for manipulating graphs, plus two clients, make and trip, that use the package. In the graph package, I implemented depth first and breadth first search algorithms. The trip client was a simplified version of a GPS program that uses A* search to find the shortest path between two points. Make was an implementation of the Unix “make” command.

Jumping Cubes
November 2013
Written in Java, I created a game involving a board, where each square has a dot value, which can be played against a human player or another computer program (AI) . I used a minimax algorithm, which employed alpha-beta pruning, in a search tree of moves to find the optimal move.
I also created a GUI for 2 player games where the board could be resized and multiple games could be played in the same session.

Image Recognition
February 2014
Written in C, we created a program that took in an image and returned what digit the image is by comparing the image with a set of template number images. Template images were flipped, rotated, transposed, and translated onto the given image to determine the closest match.

Connect Four MapReduce
March 2014
Written in Java. Constructed a full game tree for connect four, and then solved the game by running minimax to label each game state with a win, loss or tie value.
Beginning with an empty board, all possible moves and their subsequent board configurations were generated. With the generated game tree, the optimal move for the player was chosen according to the minimax algorithm. This program implemented two MapReduce jobs to accomplish solving the games.
